SamkaryaUI Notify

A powerful, flexible, and easy-to-use notification system

Installation

Include the notification system in your project:

<script type="module">
    import Notify from './samkaryaUiNotify.js';
</script>

Basic Usage

Simple Notifications

Notify.show({
    title: 'Hello World',
    message: 'This is a basic notification',
    type: 'info' // 'success', 'error', 'warning'
});

Advanced Features

Notifications with Actions

Notify.show({
    title: 'Confirm Action',
    message: 'Are you sure you want to proceed?',
    type: 'warning',
    duration: 0,
    actions: [
        {
            label: 'Yes',
            onClick: () => console.log('Confirmed'),
            dismiss: true
        },
        {
            label: 'No',
            onClick: () => console.log('Cancelled'),
            dismiss: true
        }
    ]
});

Progress Bar

Notify.show({
    title: 'Downloading...',
    message: 'File download in progress',
    type: 'info',
    progress: true,
    duration: 5000
});

API Reference

Configuration Options

Property Type Default Description
title string '' The title of the notification
message string '' The main message content
type string 'info' One of: 'info', 'success', 'error', 'warning'
position string 'top-right' Position of the notification
duration number 5000 Duration in ms (0 for no auto-dismiss)
progress boolean false Show progress bar
actions array [] Array of action buttons
pauseOnHover boolean true Pause timer on hover
